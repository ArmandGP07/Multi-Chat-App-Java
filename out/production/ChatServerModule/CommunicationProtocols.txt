User & Server interactions (commands)

1. User --> Server

    * login / logoff
    * status

2. Server --> User

    * online / offline

3. User --> User

    * direct messages
    * broadcast messages

COMMANDS:

    login <user> <password>
    logoff

    message || msg <user> body....
    i.e.: guest: "msg armando Hello world"  <-- sent
          armando: "msg guest Hello world"  <-- received
